## Installation
                      
Simply visit the [Heroku page](https://protected-waters-91627.herokuapp.com/). You can also run the app locally by visiting the [GitHub page](https://github.com/markmckenna37/Express-Handlebars-Burgers) and copy the link under the "code" button. Then, with your GitBash or Terminal, type "git clone" and paste the link into the field. Then, type "npm init" and "npm install" to get all of the required files.
                      
                      
---
                      
## Usage 
                      
Simply type your desired burger in the add burger input. Then, delete/devour at your leisure.
